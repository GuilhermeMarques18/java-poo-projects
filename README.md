A collection of Java projects focused on **Object-Oriented Programming (OOP)**, developed during my **3rd semester** of Software Engineering.

---

### 🎯 Core Principles Explored
This repository serves as a practical lab for mastering:

* **🛡️ Encapsulation:** Data protection and controlled access.
* **🧬 Inheritance:** Class hierarchies and code reuse.
* **🎭 Polymorphism:** Dynamic method binding and interfaces.
* **🧱 Abstraction:** Defining blueprints with Abstract Classes and Interfaces.

---

### 📂 Repository Structure
The projects are organized by specific OOP concepts:

| Package | Description |
| :--- | :--- |
| `agregacao_simples` | Object composition and association logic. |
| `array_list` | Dynamic array handling and manipulation. |
| `interfaces` | Contracts, decoupling, and multiple implementations. |
| `maps` | Key-value pair data structures. |
| `ordenacao` | Custom sorting logic and algorithms. |
| `modificadoresAcesso` | Visibility levels (`private`, `protected`, `public`). |
